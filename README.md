# 🗡️ Dungeon Crawler RPG

A terminal-based RPG game written in C with beautiful ANSI-colored UI.

## ✨ Features

- 🎮 6 interconnected dungeon rooms to explore
- ⚔️ Turn-based combat system with 3 unique enemies
- 🎒 Inventory system with items and equipment
- 📈 Experience points and leveling system
- 💰 Gold collection and treasure hunting
- 🎨 Beautiful ANSI-colored terminal interface
- 🏆 Victory condition: Defeat the Ancient Dragon!

## 🚀 How to Build & Run

```bash
# Compile the game
gcc dungeon.c -o dungeon

# Run the game
./dungeon
